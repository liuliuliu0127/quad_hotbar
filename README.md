# Quad Hotbar
Modified double hotbar mod that expands the HUD for more than 2 rowss of hotbar,.
Currently supports Minecraft 1.17.1-1.21.10

# Features
1. Visually displays more rows of the inventory on the HUD. Health bar and experience bar are moved up to make space for the new "hotbar".
2. Adds a keybind that allows the user to swap specfic rows of the inventory and the hotbar (Default: R)
3. 1.18.2+ only: A single stack can be swapped by double tapping the slot number or by holding the swap key (Default: R)
